# backend-phonedyguard
## ✏️프로젝트 개요
사회적 약자를 위한 위치추적 어플

보호자들이 미취학 아동, 치매 환자 등의 사회적 약자들이 실종되는 것에 대한 걱정을 덜어주기 위해 프로젝트를 시작했습니다.
- 실시간으로 피보호자의 위치를 추적할 수 있습니다.
- 안심경로를 미리 등록하여 사회적 약자가 경로를 이탈했을 경우 알림을 보내어 경고를 주고, 피보호자의 위치 기록을 남깁니다.
- 게시판을 통해 지역 정보를 공유합니다.
